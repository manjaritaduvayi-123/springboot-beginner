# Random Profile Generator

A simple Spring Boot + HTML/CSS/JavaScript project that generates random user profile cards.

## Features
- Random profile generator
- Toggle between male and female profiles
- Responsive profile card UI
- Pure HTML, CSS, and JavaScript
- Spring Boot backend

## Technologies Used
- Spring Boot
- HTML
- CSS
- JavaScript
